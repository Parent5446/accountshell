accountshell by Parent5446

Python-based Account Creation Shell

Account Shell takes requests from users to obtain an account from the server it is installed on for whatever intended purpose. Currently only SSH or similar command line interface is supported for creating, deleting, and approving requests. Future front-ends will be implemented. CURRENTLY IN DEVELOPMENT

Dependencies

Depends on python.

Install

1. Create a new user account named acctshell with the password acctshell and its home directory in /opt/acctshell. 2. Extract the source into /opt/acctshell and set the user's login shell to /opt/acctshell/acctshell.py. 3. The acctshell user must be given sudo privileges to the useradd command. 4. It is recommended that the permissions on /opt/acctshell be set as restrictive as possible.
